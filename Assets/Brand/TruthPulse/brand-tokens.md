# TruthPulse Brand Tokens

## Brand core

- Name: `TruthPulse`
- Domain: `truthpulse.co`
- Logo direction: `Equalizer`
- Positioning: fast, signal-forward, market-aware, native utility

## Color palette

- Primary mint: `#10C58A`
- Deep mint: `#08966B`
- Ink: `#10201B`
- Muted slate: `#5D6C67`
- Surface: `#F4FBF7`
- Surface alt: `#E8F7EF`
- White: `#FFFFFF`

## Usage

- Use `Primary mint` for the pulse/equalizer mark and key interactive moments.
- Use `Ink` for wordmarks, high-contrast text, and menu bar monochrome contexts.
- Use `Deep mint` for hover/active accents and darker backgrounds.
- Prefer soft off-white surfaces over pure white when presenting the logo in-product.

## Typography

- Primary UI / wordmark companion: `SF Pro Display`
- Web fallback: `Inter`, `system-ui`, `sans-serif`
- Brand voice: high-clarity, compact, slightly premium, no decorative serif

## Mark system

- `truthpulse-icon.svg`
  Primary app icon / square mark.
- `truthpulse-icon-dark.svg`
  Dark-mode square mark.
- `truthpulse-logotype.svg`
  Primary horizontal lockup.
- `truthpulse-stacked.svg`
  Editorial / launch / splash use.
- `truthpulse-menubar.svg`
  Compact monochrome menu bar lockup.
- `truthpulse-monochrome.svg`
  One-color brand lockup for constrained contexts.

## Rationale

The selected `Equalizer` direction was translated into a single pulse/equalizer stroke instead of literal bars. This keeps the brand readable at very small sizes, makes the mark feel more native to a menu bar utility, and avoids looking like generic audio software.
