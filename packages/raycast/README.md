# TruthPulse — Kalshi Markets

Instant search across all open Kalshi prediction markets with live odds and trend data.

## Features

- Search 10,000+ open Kalshi prediction markets by title, category, or outcome
- Live odds displayed inline for every result
- Trend data (1D / 7D / 30D) with price delta
- Local cache for instant results, background sync with Kalshi API
- Open any market on kalshi.com with one keystroke

## Commands

| Command | Description |
|---------|-------------|
| Search Kalshi Markets | Search all open prediction markets with live odds |
